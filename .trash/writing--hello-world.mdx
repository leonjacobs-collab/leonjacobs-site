## The rebuild

This site has been rebuilt from scratch with **Next.js**, **MDX**, and a design system
rooted in retrofuturism and neumorphism — warm near-black backgrounds, orange accents,
and soft extruded shadows that make every element feel like it's carved from the screen.

## Why MDX?

MDX lets me write prose in Markdown while embedding React components inline.
Code blocks get syntax highlighting via `rehype-pretty-code` and Shiki:

```typescript
const message: string = "hello from the new site";
console.log(message);
```
